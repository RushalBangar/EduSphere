# EduSphere AI 🌌

An intelligent, cinematic-grade 3D Study & Flashcard Workspace designed to maximize learning efficiency. Master key syllabus decks with dynamic interactive flips, fully-integrated visual reference sheets, high-fidelity lightboxes, and a reactive AI Explanations engine.

Built with passion and premium visual engineering by **Rushal Bangar**.

---

## ✨ Features & Visual Highlights

### 1. 🌌 Ambient Interactive Canvas Particle System
* **Constellation Connectivity**: Floating, glowing particle nodes connected by elastic digital lines that dynamically fade based on proximity.
* **Interactive Physics**: Particles actively respond to user mouse pointers, creating a responsive and living space.
* **Dual-Theme Adaptation**: Deep cyber-obsidian tones with vibrant particles in dark mode, and soft light-slate tones in light mode.
* **Multi-Layered Auroras**: 4 separate floating background gradient nodes cycling colors dynamically.

### 2. 💎 Premium Orbiting Logo & Header Branding
* **E-Monogram Symbol**: Custom geometric SVG logo mark representing "EduSphere" inside a modern glassy medallion.
* **3D Elliptical Orbit**: A glowing cyan satellite sphere loops smoothly around the main structure.
* **Badge Glint**: A bright translucent glint shimmers across the "AI" label badge every few seconds.

### 3. 🃏 High-Fidelity 3D Flashcards
* **Y-Axis Rotation**: Fluid, hardware-accelerated 3D transforms for flipping cards to inspect answers.
* **Interactive Syllabus Panels**: Dynamically mounts relevant engineering syllabus maps (FDS & OOPs diagrams) directly within the dashboard.
* **High-Res Lightbox**: Fullscreen cinematic overlay with smooth backdrop-dismissal and zoom transition features.

### 4. 🧠 Dynamic AI explanations Drawer
* **Double-Sheet Panel**: Adapts from a right-hand sidebar on desktop to a bottom drawer on mobile viewports.
* **Syllabus Context**: Uses NotebookLM explanations database to explain tricky terms instantly.

## 🛠️ Technology Stack
* **Core Structure**: HTML5 Semantic Architecture.
* **Design Styling**: Custom CSS Variables, keyframes, transitions, and radial gradient blends.
* **Interactive Engine**: Vanilla JavaScript (Canvas API, DOM manipulation, SMIL SVG vector paths).

## 🚀 How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/RushalBangar/EduSphere.git
   ```
2. Open `index.html` directly in any web browser, or serve it using a lightweight dev server (like Live Server in VS Code):
   ```bash
   npx serve .
   ```
